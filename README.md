# NarrativeX Agent

AI agent for detecting narrative rotations, assessing market risk, and generating portfolio recommendations from CoinMarketCap intelligence.

## Problem

Crypto traders struggle to identify when capital is rotating from one narrative to another. The signal is fragmented across categories, price momentum, volume expansion, and regime changes, so by the time a story is obvious, the move is often late.

## Solution

NarrativeX Agent uses CoinMarketCap market intelligence to detect dominant narratives, evaluate risk, and generate portfolio recommendations. It combines category data, narrative strength, market regime, and rotation scoring into a single decision layer that helps traders understand what is leading, what is fading, and how to position around it.

## Features

- Narrative Detection
- Narrative Rotation Agent
- Market Regime Analysis
- Risk Engine
- Allocation Engine
- Decision Log
- Architecture Viewer
- CoinMarketCap Integration

## How It Works

CoinMarketCap Data  
→ Narrative Detection  
→ Rotation Scoring  
→ Risk Analysis  
→ Narrative Rotation Agent  
→ Portfolio Recommendations
